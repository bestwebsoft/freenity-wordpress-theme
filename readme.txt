== Description ==

Freenity is an excellent choice for photographers, freelancers, startups, personal portfolios, landing pages and blogs. Its stylish and modern design is the best way to stand out from the crowd and get more visitors from Google and other search engines. Theme is also powered by Font Awesome. Freenity is simple, easy to use and fast. This professional and elegant theme is great for anyone looking to display a lot of content in a userfriendly and clean manner. Live demo: http://wp-demo-theme.bestweblayout.com/freenity/

= Features =

* Cross-browser compatible
* Two-column design
* Fixed-layout
* Threaded comments
* Translation-ready
* Pagination
* Main colors: white, blue, gray

== Installation ==

Manual installation:

1. Upload the "Freenity" folder to “/wp-content/themes/” directory.

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New.
2. Search for "Freenity".
3. Click "Install" button to open the theme's repository listing.
4. Click "Install" button.

Activation and Use

1. Activate the Theme through "Themes" menu in WordPress

== Theme Notes ==

= Post Formats: Aside, Audio, Chat, Gallery, Image, Link, Quote, Status, Video =

Freenity WordPress theme supports only these post formats. Please style them in a certain way. Other post formats will be displayed as regular posts.

= Widgets =

The Theme includes a standard Widget. The following widgets are shown by default: recent posts, recent comments, archives, categories.
Also added to this topic widgets such as Freenity Description Text - Freenity Text widget with Logo footer, Freenity Recent Posts - Widget recent posts for freenity theme display featured image, Freenity Tabber Widget - Simple jQuery Text Tabber Widget.

== Frequently Asked Questions ==

= How to display featured image? =

You can do this in Posts section > Add new (or edit the existing one) > Set featured image.

= How to use custom-background? =

You can easily change background color via Admin UI (Dashboard) > Appearance > Customize > Colors > Background color.

= How can I change language from English into Russian? =

Change the language in the admin settings screen. Settings > General > Site Language.
